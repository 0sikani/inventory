# Inventory API

## System Requirements
- **Java**: Version 25 or later
- **Database**: MySQL 8.0+
- **Operating System**: Windows 10/11, Linux, macOS

## Installation
1. Extract the distribution package
2. Configure database in `config/application-prod.properties`
3. Run `scripts/start-backend.bat` (Windows) to start the server

## API Endpoints
- Students: `GET/POST/PUT/DELETE /api/students`
- Academic Records: `GET/POST/PUT/DELETE /api/academics`
- Requests: `GET/POST/PUT/DELETE /api/requests`

## Default Port
The API runs on port 8080 by default.
